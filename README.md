# Traiings-Backoffice
Ein eigenes Tool zur Organisation und Verwaltung von Trainings der versciedenen Akademien, Locations, Trainer, CRM der potentiellen Kunden Newsletter Abos. Auch listen darüber wann welches training wo stattfindet mit checklisten für einladungen verschickt, location gebucht, lunch organisiert und weitere Schritte. Irgendwann anzubinden an Kalender
